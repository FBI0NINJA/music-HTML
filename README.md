# 🎵 Vibes Player

A modern and stylish web-based music player built with pure HTML, CSS, and JavaScript.

## ✨ Features

* 🔍 Search and load YouTube songs
* ▶️ Play / Pause controls
* ⏭ Next & Previous seek controls
* 🔊 Volume control
* 📊 Real-time progress bar
* 🎨 Animated spinning music artwork
* 🌈 Glassmorphism modern UI
* ⚡ Lightweight and fast
* 📱 Responsive design
* 🎧 Built using YouTube IFrame API

---

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript
* YouTube IFrame API
* Tabler Icons

---

## 📸 Preview

Modern neon-style music player with animated visuals and smooth controls.

---

## 📂 Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/vibes-player.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

---

## 🎯 Usage

1. Paste a YouTube video link.
2. Press the search button.
3. Enjoy your music.

Example:

```text
https://www.youtube.com/watch?v=dQw4w9WgXcQ
```
